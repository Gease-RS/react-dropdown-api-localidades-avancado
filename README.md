# Passo a passo terminal

1. Iniciando projeto do zero

```bash
    npx react-create-app react-dropdown-api-localidades
```

2. Entre no projeto criado

```bash
    cd react-dropdown-api-localidades
```

3. Comando para rodar aplicação

```bash
    npm start
```
